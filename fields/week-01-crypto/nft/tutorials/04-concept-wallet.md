# 钱包 (Wallet)

## 一句话大白话

**钱包是你在区块链世界的"身份证 + 银行卡"——用来证明你是谁，以及管理你的资产。**

但要注意：钱包不是真的"装钱"，它装的是你的"钥匙"（私钥）。你的资产其实存在区块链上，钱包只是让你能够访问和控制它们。

---

## 它解决什么问题

### 核心痛点：区块链上如何证明"你是你"

传统世界：
- 银行通过身份证、密码验证你是账户主人
- 网站通过用户名、密码验证你的身份

区块链世界没有银行、没有服务商，怎么验证身份？

### 钱包的解决方案：密码学身份

```
私钥（只有你知道）
    │
    ▼ 通过数学算法生成
公钥
    │
    ▼ 通过哈希算法生成
地址（公开的，像银行账号）
```

- **私钥**：超长密码，知道它就能控制资产（绝对保密）
- **公钥**：从私钥推导出来，用于验证签名
- **地址**：从公钥推导出来，可以公开分享（别人给你转账用）

---

## 什么时候用 / 什么时候别用

### ✅ 必须用钱包的场景

- 接收 NFT 或加密货币
- 铸造或购买 NFT
- 与任何 DApp（去中心化应用）交互
- 证明你是某个 NFT 的所有者

### ❌ 不需要钱包的场景

- 只是浏览 NFT（不购买、不交互）
- 在中心化交易所交易（交易所托管钱包）
- 只是了解学习（可以先不创建）

---

## 它不是什么（常见混淆点）

### ❌ 钱包 ≠ 存钱的地方

你的 ETH、NFT 不是存在钱包软件里，而是存在区块链上。钱包只是"钥匙"，让你能访问链上的资产。

```
错误理解：
钱包 APP 像保险箱，删了 APP 钱就没了

正确理解：
钱包 APP 像钥匙串
- 删了 APP，钥匙还在（如果你备份了助记词）
- 用钥匙可以在任何钱包 APP 里恢复
- 钱还在链上，从来没进过你的手机
```

### ❌ 钱包地址 ≠ 私钥

| | 地址 | 私钥 |
|---|------|------|
| 能公开吗 | ✅ 可以 | ❌ 绝对不能 |
| 作用 | 收款、展示 | 控制资产 |
| 类比 | 银行账号 | 银行密码 |
| 泄露后果 | 无（可能收到垃圾） | 资产全部被盗 |

### ❌ 热钱包 ≠ 冷钱包

| 类型 | 热钱包 | 冷钱包 |
|------|--------|--------|
| 定义 | 联网的钱包 | 不联网的钱包 |
| 例子 | MetaMask、手机钱包 | Ledger、纸钱包 |
| 便利性 | 高，随时可用 | 低，需要额外操作 |
| 安全性 | 相对低 | 高 |
| 适合 | 日常小额交易 | 大额长期存储 |

## 最小例子

### MetaMask 钱包结构

```
MetaMask 钱包
│
├── 账户 1 (Account 1)
│   ├── 地址: 0x1234...5678

│   └── NFT: [CryptoPunk #1234, BAYC #5678]
│
├── 账户 2 (Account 2)
│   ├── 地址: 0xabcd...efgh
│   ├── ETH 余额: 0.1 ETH
│   └── NFT: [...]
│
└── 助记词 (Seed Phrase): 12/24 个单词
    → 可以恢复所有账户
```

### 一个地址长什么样

以太坊地址示例：
```
0x71C7656EC7ab88b098defB751B7401B5f6d8976F
│ │                                        │
│ └────────── 40 个十六进制字符 ──────────────┘
│
└── 固定前缀 "0x"
```

### 助记词长什么样

```
助记词示例（12 个单词）：
apple banana cherry dog elephant frog guitar horse ice juice kiwi lemon

⚠️ 警告：
- 这是你的"超级密码"
- 知道助记词 = 完全控制所有资产
- 绝对不要告诉任何人
- 绝对不要输入到任何网站
- 绝对不要截图或在线存储
```

## 新手最常踩的 3 个坑

### 1. 助记词泄露

**坑**：把助记词存在微信、截图、云盘、或告诉"客服"

**后果**：资产秒被转走，无法追回

**正确做法**：
- 手写在纸上，存放安全位置
- 可以刻在金属板上（防火防水）
- 永远不要数字化存储

### 2. 连接钓鱼网站

**坑**：点击 Discord/推特上的链接，连接钱包

**后果**：
- 恶意合约获得授权
- 资产被静默转走

**正确做法**：
- 手动输入官网地址
- 检查 URL 是否正确
- 不点击陌生链接
- 使用专门的"小号钱包"进行测试

### 3. 误以为删了 APP 钱没了

**坑**：删了钱包 APP，以为钱没了

**真相**：
- 只要有助记词，可以在任何钱包恢复
- 钱一直在链上，从来不在你手机里

**正确做法**：
- 创建钱包时务必备份助记词
- 恢复钱包只需要助记词

└──────────┘    └──────────┘    └──────────┘    └──────────┘    └──────────┘
     │                               │               │               │
     ▼                               ▼               ▼               ▼
 【创建钱包】                    【连接钱包】     【签名交易】     【查看资产】
 这是第一步！                   授权平台操作     确认买卖         NFT 显示在钱包里

钱包贯穿 NFT 的整个生命周期：
- **准备阶段**：创建钱包，获取 ETH
- **铸造阶段**：连接钱包到铸造网站
- **交易阶段**：签名确认交易
- **持有阶段**：NFT 显示在钱包中

## 实操指南：创建 MetaMask 钱包

### 第一步：安装
1. 访问 https://metamask.io（手动输入，别点链接）
2. 下载浏览器扩展或手机 APP

### 第二步：创建新钱包
1. 点击"创建新钱包"
2. 设置登录密码（这只是本地密码）
3. **最重要**：备份助记词
   - 按顺序写在纸上
   - 验证你记对了

### 第三步：获取测试币
1. 切换到测试网（Sepolia）
2. 访问水龙头网站获取免费测试币
3. 用测试币练习操作

## 自测题

1. **概念理解**：如果你删掉了 MetaMask APP，你的 NFT 会消失吗？为什么？

2. **安全意识**：以下哪些行为是危险的？
   - A. 把钱包地址发给朋友让他给你转 NFT
   - B. 把助记词存在 iCloud 备忘录里
   - C. 在新网站上连接钱包前先查证网站真伪
   - D. 在推特私信里输入助记词"验证身份"

3. **实操判断**：你有 1 ETH 想长期持有，有 0.1 ETH 经常用来买 NFT，应该怎么存放？

<details>
<summary>参考答案</summary>

1. 不会消失。NFT 存在区块链上，不是存在你的手机里。只要你备份了助记词，就可以在任何钱包 APP（MetaMask、Rainbow、Trust Wallet 等）里恢复，看到你的所有资产。删 APP 只是删了"钥匙串"软件，"钥匙"（助记词）和"资产"（链上数据）都还在。

2. 危险的是 B 和 D。
   - A：安全，地址可以公开，就像银行账号
   - B：极度危险！云存储可能被黑客攻破
   - C：正确做法
   - D：极度危险！这是典型骗局，没有任何正规服务需要你的助记词

3. 建议分开存放：
   - 1 ETH 放冷钱包（Ledger）或另一个不常用的热钱包地址
   - 0.1 ETH 放 MetaMask 日常使用
   - 这样即使日常钱包被钓鱼，损失也有限

</details>
