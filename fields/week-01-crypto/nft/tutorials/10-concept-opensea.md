# OpenSea

## 一句话大白话

**OpenSea 是 NFT 世界的"淘宝"——最大的 NFT 交易市场，可以买卖、浏览、创建 NFT。**

就像淘宝连接买家和卖家，OpenSea 连接 NFT 的创作者、卖家和买家，提供一个方便的交易平台。

---

## 它解决什么问题

### 核心痛点：NFT 交易太复杂

如果没有交易市场：
- 买家要找卖家谈判
- 需要手动调用合约转账
- 不知道当前市场价格
- 容易被骗（先付钱不发货？先发货不收到钱？）

### OpenSea 的解决方案

```
传统方式：
买家 ←→ 私下联系 ←→ 卖家
       ↓
    手动转账，风险自担

OpenSea 方式：
买家 → OpenSea 智能合约 ← 卖家
         ↓
    自动撮合，安全交易
```

OpenSea 提供：
- **浏览**：看到所有上架的 NFT
- **搜索**：按项目、属性、价格筛选
- **定价**：看到地板价、历史成交
- **交易**：一键购买或出价
- **安全**：智能合约保证交易原子性（要么都成功，要么都失败）

---

## 什么时候用 / 什么时候别用

### ✅ 适合用 OpenSea 的场景

- 购买或出售 NFT
- 浏览和研究 NFT 项目
- 创建和铸造简单的 NFT
- 查看 NFT 的历史价格和交易记录

### ❌ 不适合的场景

- 参与项目首发铸造（通常在项目官网）
- 追求最低手续费（可能有更便宜的平台）
- Solana 等非 EVM 链的 NFT（OpenSea 支持有限）
- 需要高级交易功能（专业工具如 Blur）

---

## 它不是什么（常见混淆点）

### ❌ OpenSea ≠ NFT 本身

OpenSea 是交易平台，不是 NFT 的存储位置：
- NFT 存在区块链上
- OpenSea 只是读取链上数据显示出来
- 不在 OpenSea 上架，NFT 也存在
- OpenSea 关闭，NFT 还在

### ❌ OpenSea ≠ 唯一的交易平台

| 平台 | 特点 | 手续费 |
|------|------|--------|
| OpenSea | 最大，最全面 | 2.5% |
| Blur | 专业交易者，无手续费 | 0% |
| LooksRare | 有代币激励 | 2% |
| X2Y2 | 费用较低 | 0.5% |
| Magic Eden | Solana 生态主流 | 2% |

### ❌ 在 OpenSea 看到 ≠ 一定真

- 任何人都可以铸造 NFT 假装是知名项目
- OpenSea 有验证标志（蓝勾），但也可能有遗漏
- 要核对合约地址是否正确

## 新手最常踩的 3 个坑

### 1. 买到假货

**坑**：搜索某个知名项目，买到同名的假 NFT

**如何避免**：
- 检查是否有蓝色验证标志 ✓
- 核对合约地址与官方公布的是否一致
- 查看交易量和持有人数（假货通常很少）
- 不通过搜索，直接从官方链接进入

**对比**：
```
真 BAYC：
- 合约: 0xBC4CA0EdA7647A8aB7C2061c2E118A18a936f13D
- 持有人: 10,000+
- 交易量: 数十万 ETH

假 BAYC：
- 合约: 0x1234...（随便的地址）
- 持有人: 几十个
- 交易量: 几乎为 0
```

### 2. 不理解"上架"和"授权"的成本

**坑**：以为上架免费，结果要付 Gas

**解释**：
- **首次出售**：需要 approve 给 OpenSea，要付 Gas
- **创建订单**：有时需要付 Gas（链上订单）
- **取消订单**：需要付 Gas

**省钱技巧**：
- 使用 Polygon 版本（Gas 极低）
- 选择低峰期操作
- 一次多上架几个 NFT

### 3. 收到"空投"NFT 不要动

**坑**：钱包里突然出现不认识的 NFT，想卖掉

**风险**：
- 可能是钓鱼 NFT
- 交互（上架/转账）可能触发恶意合约
- 可能导致资产被盗

**正确做法**：
- 不明 NFT 不要碰
- 不要尝试上架或转账
- 可以直接忽略或隐藏

## OpenSea 实用功能

### 1. 集合页面分析

```
集合统计：
- Floor Price（地板价）：最便宜的在售价格
- Volume（交易量）：总交易额
- Owners（持有人数）：多少人持有
- Items（总量）：NFT 总数
- Listed（上架数）：正在出售的数量
```

### 2. 高级筛选

```
可筛选条件：
- 价格范围
- 属性（背景、服装、配饰等）
- 状态（在售、有报价、新上架）
- 币种（ETH、WETH）
```

### 3. 活动追踪

```
可追踪的活动：
- 销售（Sales）
- 上架（Listings）
- 报价（Offers）
- 转账（Transfers）
```

## 自测题

1. **概念理解**：如果 OpenSea 网站关闭了，你在上面买的 NFT 会消失吗？为什么？

2. **安全意识**：你搜索"Bored Ape"想买一个，看到两个集合，一个有蓝勾一个没有，价格差 100 倍。应该怎么判断？

3. **成本计算**：你想在 OpenSea 上卖一个价值 1 ETH 的 NFT，最终能拿到多少钱？（假设 OpenSea 费率 2.5%，创作者版税 5%）

<details>
<summary>参考答案</summary>

1. 不会消失。NFT 存在区块链上，不存在 OpenSea 的服务器上。OpenSea 只是读取链上数据并显示出来的"前端"。如果 OpenSea 关闭：
   - 你的 NFT 还在你的钱包里
   - 可以在其他平台（Blur、LooksRare 等）看到和交易
   - 链上数据不受影响

2. 判断方法：
   - 蓝勾是一个信号，但不是唯一标准
   - 核对合约地址：去 BAYC 官网或官方推特找到正确的合约地址
   - 查看数据：真品应该有大量交易历史、数千持有人
   - 价格逻辑：如果一个"正品"价格异常低，几乎肯定是假的

3. 计算：
   - 售价：1 ETH
   - OpenSea 费用：1 × 2.5% = 0.025 ETH
   - 创作者版税：1 × 5% = 0.05 ETH
   - 你拿到：1 - 0.025 - 0.05 = 0.925 ETH

   注意：实际交易中可能还需要考虑 Gas 费用

</details>
