# 实战案例：在测试网铸造你的第一个 NFT

## 案例背景

**目标**：零成本（不花真钱）完成一次完整的 NFT 铸造流程

**为什么用测试网**：
- 测试网的 ETH 是免费的（从"水龙头"领取）
- 操作和主网完全一样
- 犯错了不会损失真金白银
- 学完后可以无缝切换到主网

**我们要做什么**：
1. 创建钱包
2. 获取测试币
3. 准备图片和元数据
4. 上传到 IPFS
5. 在 OpenSea 测试网铸造 NFT
6. 查看并分享你的 NFT

---

## 流程演练

### 节点 1：创建 MetaMask 钱包

**输入**：
- 一台电脑或手机
- Chrome/Firefox/Edge 浏览器

**加工步骤**：

```
1. 访问 https://metamask.io
   ⚠️ 手动输入网址，不要点击任何链接！
   ⚠️ 确认地址栏显示的是 metamask.io

2. 点击 "Download" 下载浏览器扩展

3. 安装完成后，点击浏览器右上角的 MetaMask 图标

4. 选择 "创建新钱包"

5. 设置密码（这是本地登录密码）
   - 至少 8 个字符
   - 建议包含大小写字母和数字

6. 【最重要】备份助记词
   - 会显示 12 个英文单词
   - 拿纸笔按顺序抄下来
   - 不要截图！不要存到电脑！
   - 这 12 个单词 = 你资产的全部控制权

7. 验证助记词（按顺序点选）

8. 完成！你会看到钱包主界面
```

**输出**：
- 一个可用的 MetaMask 钱包
- 一个钱包地址（形如 `0x1234...5678`）
- 12 个助记词（安全保存）

**常见失败原因**：
1. 下载到假的 MetaMask（从非官网下载）→ 只从官网下载
2. 助记词丢失 → 立即用纸笔抄写
3. 助记词被他人看到 → 在私密环境下操作

**检查点** ✓
- [ ] MetaMask 图标出现在浏览器工具栏
- [ ] 能看到自己的钱包地址
- [ ] 助记词已安全记录在纸上

---

### 节点 2：切换到测试网并获取测试币

**输入**：
- 已安装的 MetaMask 钱包
- 钱包地址

**加工步骤**：

```
1. 切换到测试网络

   方法 A（推荐 - Sepolia 测试网）：
   - 点击 MetaMask 左上角的网络下拉菜单
   - 点击 "显示测试网络"
   - 选择 "Sepolia"

2. 获取测试 ETH（从水龙头）

   访问以下任一水龙头网站：
   - https://sepoliafaucet.com/
   - https://www.alchemy.com/faucets/ethereum-sepolia
   - https://sepolia-faucet.pk910.de/

3. 操作步骤（以 sepoliafaucet 为例）：
   - 复制你的钱包地址（点击 MetaMask 中的地址即可复制）
   - 粘贴到水龙头网站的输入框
   - 完成验证（可能需要登录或验证码）
   - 点击 "Send" 或 "Request"

4. 等待到账
   - 通常几秒到几分钟
   - MetaMask 会显示余额变化
```

**输出**：
- 钱包显示在 Sepolia 测试网
- 余额显示测试 ETH（通常 0.05-0.5 ETH）

**常见失败原因**：
1. 还在主网（Ethereum Mainnet）→ 确认切换到 Sepolia
2. 水龙头被用完 → 换一个水龙头试试
3. 地址复制错误 → 确认地址格式正确（0x 开头，42 个字符）

**检查点** ✓
- [ ] MetaMask 显示 "Sepolia 测试网络"
- [ ] 余额 > 0 ETH

---

### 节点 3：准备图片和元数据

**输入**：
- 一张图片（你自己创作的）
- 关于这个 NFT 的描述

**加工步骤**：

```
1. 准备图片
   - 格式：PNG、JPG、GIF、SVG 都可以
   - 大小：建议 1000x1000 像素左右
   - 注意：必须是你自己创作的，不要侵权！

   示例：用画图软件画一个简单图案

2. 思考元数据内容
   - 名称：给你的 NFT 起个名字
   - 描述：简单描述这是什么
   - 属性（可选）：比如颜色、风格等

   示例：
   名称：My First NFT
   描述：这是我在学习 NFT 时创建的第一个作品
   属性：
   - 背景：蓝色
   - 风格：简约
```

**输出**：
- 一张准备好的图片文件
- 想好的名称、描述、属性

**常见失败原因**：
1. 图片太大（超过 100MB）→ 压缩图片
2. 使用他人作品 → 只用自己的创作
3. 不支持的格式 → 转换为 PNG/JPG

**检查点** ✓
- [ ] 图片已准备好
- [ ] 名称和描述已想好

---

### 节点 4：上传到 IPFS

**输入**：
- 准备好的图片
- 元数据内容

**加工步骤**：

```
方法 A：使用 Pinata（推荐新手）

1. 访问 https://www.pinata.cloud/
2. 注册账号（免费）
3. 登录后，点击 "Upload" → "File"
4. 上传你的图片
5. 上传成功后，复制 CID（形如 QmXxx...）
6. 图片的 IPFS 链接就是：ipfs://[你的CID]

方法 B：使用 NFT.Storage（完全免费）

1. 访问 https://nft.storage/
2. 使用 GitHub 或邮箱注册
3. 点击 "Upload" 上传图片
4. 获取 CID

记录下来：
- 图片 CID：QmXxx...
- 图片 IPFS 链接：ipfs://QmXxx...
```

**输出**：
- 图片已上传到 IPFS
- 获得图片的 CID 和 IPFS 链接

**常见失败原因**：
1. 上传失败 → 检查网络，文件是否太大
2. 找不到 CID → 在 "Files" 或 "Uploads" 列表中查看
3. 链接格式错误 → 确认是 `ipfs://QmXxx...` 格式

**检查点** ✓
- [ ] 图片已上传到 IPFS
- [ ] 已记录图片的 IPFS 链接

---

### 节点 5：在 OpenSea 测试网铸造 NFT

**输入**：
- MetaMask 钱包（已有测试 ETH）
- 图片的 IPFS 链接
- NFT 的名称和描述

**加工步骤**：

```
1. 访问 OpenSea 测试网
   https://testnets.opensea.io/

2. 连接钱包
   - 点击右上角钱包图标
   - 选择 MetaMask
   - 在 MetaMask 弹窗中确认连接

3. 创建 NFT
   - 点击右上角头像 → "Create"
   - 或直接访问 https://testnets.opensea.io/collection/create

4. 填写 NFT 信息
   - Image: 可以上传图片，或使用你的 IPFS 链接
   - Name: 输入 NFT 名称
   - External link: 可选
   - Description: 输入描述
   - Collection: 选择或创建一个集合
   - Properties/Levels/Stats: 添加属性（可选）

5. 点击 "Create"

6. 签名确认
   - MetaMask 会弹出签名请求
   - 确认后完成铸造
   - 注意：在 OpenSea 测试网上创建可能不需要 Gas
```

**输出**：
- 你的第一个 NFT 诞生了！
- 可以在 OpenSea 测试网上查看

**常见失败原因**：
1. 没连接钱包 → 确认钱包已连接
2. 网络不对 → 确认在 Sepolia 测试网
3. 签名被拒绝 → 在 MetaMask 中点击确认

**检查点** ✓
- [ ] NFT 创建成功
- [ ] 可以在 OpenSea 测试网看到你的 NFT

---

### 节点 6：查看和验证你的 NFT

**输入**：
- 已铸造的 NFT

**加工步骤**：

```
1. 在 OpenSea 查看
   - 点击右上角头像 → "Profile"
   - 在 "Collected" 标签下能看到你的 NFT

2. 查看 NFT 详情
   - 点击你的 NFT
   - 检查信息是否正确：名称、描述、图片
   - 查看 "Details" 确认合约地址和 Token ID

3. 在 MetaMask 中查看（可选）
   - 打开 MetaMask
   - 点击 "NFTs" 标签
   - 如果没显示，点击 "Import NFTs"
   - 输入合约地址和 Token ID

4. 分享你的 NFT
   - 复制 NFT 详情页的链接
   - 发给朋友炫耀一下！
```

**输出**：
- 确认 NFT 信息正确
- 可以分享的 NFT 链接

**常见失败原因**：
1. 看不到 NFT → 刷新页面，等待几分钟
2. MetaMask 不显示 → 需要手动导入
3. 图片加载不出来 → IPFS 网关可能需要时间，稍后再试

**检查点** ✓
- [ ] NFT 详情页可以访问
- [ ] 图片和信息显示正确
- [ ] 你是 NFT 的所有者（Owner）

## 常见坑点提醒

### 1. 安全相关

```
❌ 不要把助记词告诉任何人
❌ 不要在任何网站输入助记词
❌ 不要点击不明链接连接钱包
✅ 只访问官方网站
✅ 确认网址拼写正确
```

### 2. 操作相关

```
❌ 不要在主网上测试（会花真钱）
❌ 不要关闭签名确认窗口
✅ 确认在测试网操作
✅ 耐心等待交易确认
```

### 3. 内容相关

```
❌ 不要上传他人作品
❌ 不要上传违规内容
✅ 使用自己的原创内容
✅ 图片大小适中（<100MB）
```

## 复盘

### 我学到的 3 件事

1. **NFT 铸造的本质**
   - 铸造 = 在区块链上创建一条记录
   - 图片存在链外（IPFS），链上只存指针
   - 所有权记录永久保存在链上

2. **钱包的重要性**
   - 钱包是区块链世界的入口
   - 助记词 = 最高权限，丢失就无法找回
   - 不同网络的资产是独立的

3. **实际操作的成本**
   - Gas 费是每次链上操作的必要成本
   - 测试网是免费的学习环境
   - 主网操作前一定要先在测试网练习

### 仍不确定的 3 件事

1. **技术细节**
   - ERC-721 合约的完整代码如何编写？
   - 如何自己部署 NFT 合约而不是用平台？

2. **市场相关**
   - 如何定价一个 NFT？
   - 如何让更多人看到我的 NFT？

3. **进阶功能**
   - 如何设置版税？
   - 如何做动态 NFT（内容会变化的 NFT）？

### 下一步要查的 3 个点

1. **学习 Solidity**
   - 理解智能合约语言
   - 自己编写和部署 ERC-721 合约

2. **探索 Layer 2**
   - Polygon、Arbitrum 等低 Gas 方案
   - 如何在 L2 上铸造 NFT

3. **了解 NFT 项目运营**
   - 如何做一个 NFT 系列
   - 白名单机制是什么
   - 如何建立社区
