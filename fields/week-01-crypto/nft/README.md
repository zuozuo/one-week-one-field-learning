> **口诀：终局定方向，流程搭骨架，单点打通关，类比挂认知，闭环验真知。**

---

## 1. 终极目标

### 解决的问题
**数字资产的所有权确认和稀缺性证明**

在数字世界里，复制粘贴几乎零成本。一张图片、一段音乐、一个视频，传统上无法证明"谁是原始拥有者"。NFT 解决的核心问题是：**让数字内容像实物一样，有唯一的"产权证"**。

### 最终交付物
- 一个可验证的、独一无二的数字资产凭证
- 记录在区块链上，任何人可查询，无法篡改

### 评判标准
- 能在区块链浏览器上查到这个 NFT 的完整历史
- 钱包里能看到这个 NFT
- 可以在 OpenSea 等平台展示、转让或出售

### 验收任务
> **独立在测试网上铸造一个 NFT，并能解释 ERC-721 标准、Gas 费用、元数据存储方案的取舍。**

## 5. 学习自检清单

### 费曼检验
- [ ] 我能用 3 句话向朋友解释"什么是 NFT"吗？
- [ ] 我能画出 NFT 从创作到交易的完整流程吗？
- [ ] 我能解释为什么 NFT 图片"右键保存"后不等于拥有吗？

### 迁移检验
- [ ] 如果换一个区块链（如 Polygon），我知道流程有什么不同吗？
- [ ] 如果要铸造一个音乐 NFT 而不是图片，我知道怎么做吗？
- [ ] 如果朋友想卖 NFT，我能指导他完成全流程吗？

### 深度检验
- [ ] 我能说出 ERC-721 和 ERC-1155 的区别和适用场景吗？
- [ ] 我能解释 Gas 费为什么会波动、如何省 Gas 吗？
- [ ] 我能说出元数据存在链上 vs IPFS vs 中心化服务器的利弊吗？

## 6. 常见问题速查

| 问题 | 简答 |
|------|------|
| NFT 和加密货币有什么区别？ | 加密货币可互换（1 ETH = 1 ETH），NFT 不可互换（每个都独一无二） |
| NFT 图片存在哪里？ | 通常存在 IPFS 或中心化服务器，链上只存指向图片的链接 |
| 为什么有些 NFT 那么贵？ | 稀缺性 + 社区共识 + 投机 + 创作者影响力 |
| NFT 会不会归零？ | 可能，大部分 NFT 最终价值趋近于零，投资需谨慎 |
| 铸造 NFT 要花多少钱？ | 以太坊主网几美元到几十美元不等，Layer2 或其他链可能只需几分钱 |

## 7. 学习路线建议

```
第一周：理解基础
├── 理解 NFT 是什么、解决什么问题
├── 创建一个钱包（MetaMask）
└── 在测试网获取测试币

第二周：动手实践
├── 在测试网铸造第一个 NFT
├── 理解 Gas、元数据、IPFS
└── 尝试在 OpenSea 测试网上架

第三周：深入理解
├── 学习 ERC-721 标准
├── 理解智能合约基础
└── 分析几个知名 NFT 项目

第四周：进阶探索
├── 了解 ERC-1155、动态 NFT
├── 研究不同链的 NFT 生态
└── 思考 NFT 的应用场景和局限性
```

---

> **记住**：NFT 本质上是一种"数字所有权凭证"技术，真正的价值在于它被如何使用。技术只是工具，理解背后的逻辑比追逐热点更重要。
