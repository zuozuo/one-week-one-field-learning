# USDT 的一生 - 单点穿透演练

## 案例背景

我们选择 USDT（Tether）作为案例，因为它是：
- 市值最大的稳定币（超过 1000 亿美元）
- 历史最悠久（2014 年推出）
- 机制最简单直接（法币抵押型）
- 最有代表性的稳定币

通过追踪"一枚 USDT 的一生"，我们将完整理解稳定币从诞生到消亡的全过程。

## 常见坑点提醒

### 坑点 1：误以为 USDT 等于美元

```
USDT ≠ 美元

USDT 是 Tether 公司发行的"美元代金券"
它的价值取决于：
1. Tether 真的有那么多储备吗？
2. 你真的能随时换回美元吗？
3. Tether 公司会不会出问题？

类比：
游乐场的游戏币不是人民币
商场的购物卡不是人民币
USDT 也不是美元
```

### 坑点 2：忽视链的差异

```
同样是 USDT，不同链完全不同！

┌─────────────┬─────────────┬─────────────┬─────────────┐
│    链       │   特点      │   手续费    │   速度      │
```

## 复盘

### 我学到的 3 件事

1. **稳定币的本质是"信任转换"**
   - 法币稳定币：把对银行的信任转换为对发行方的信任
   - USDT 稳定的前提是 Tether 公司靠谱、储备充足

2. **锚定机制依赖多方参与**
   - 铸造/赎回：大机构参与
   - 套利：专业交易者参与
   - 普通用户：只是使用者，无法直接影响锚定

3. **"稳定"是相对的，风险是真实的**
   - 没有 100% 稳定的稳定币
   - 理解风险比追求收益更重要
   - 分散持有多种稳定币是明智之举

### 仍不确定的 3 件事

1. **Tether 的储备到底是什么？**
   - 官方说有储备，但具体构成？
   - 审计报告的可信度？
   - 如果挤兑真的发生，能 100% 兑付吗？

2. **监管将如何影响稳定币？**
   - 各国监管态度不同
   - 美国是否会立法监管？
   - 会不会要求更严格的储备证明？

3. **稳定币的未来形态？**
   - CBDC（央行数字货币）会取代稳定币吗？
   - 算法稳定币还有未来吗？
   - 下一代稳定币会是什么样？

### 下一步要查的 3 个点

1. **Tether 的审计报告**
   - 去 Tether 官网找最新的储备证明
   - 理解储备构成（现金、国债、商业票据等）
   - 对比不同时期的变化

2. **不同稳定币的对比**
   - USDT vs USDC vs DAI
   - 各自的优缺点和风险特征
   - 在不同场景下的选择建议

3. **稳定币的 DeFi 用例**
   - 稳定币借贷（Aave、Compound）
   - 稳定币流动性挖矿（Curve）
   - 收益和风险的权衡
