# 间隔重复 (Spaced Repetition)

## 一句话定义

**在你快要忘记的时候复习，用最少的时间记最久的知识。**

## 生活类比

想象你在浇花：

- ❌ **错误方式**：一天浇10次水，然后一个月不管 → 花先被淹死，后被旱死
- ✅ **正确方式**：今天浇一次，三天后浇一次，一周后浇一次 → 花健康生长

记忆也是一样：不是一口气猛学，而是**间隔着、有节奏地复习**。

## 遗忘曲线：你的记忆是怎么消失的

德国心理学家艾宾浩斯在1885年发现：

```
记忆保留率
100% │●
     │ ╲
 80% │  ╲
     │   ╲
 60% │    ╲
     │     ╲
 40% │      ╲
     │       ╲____
 20% │            ╲_____
     │                   ╲__________
  0% └──────────────────────────────────▶ 时间
     │ │    │       │           │
    刚学 20分钟 1小时    1天         1周
```

**残酷的事实**：
- 20分钟后：忘记 42%
- 1小时后：忘记 56%
- 1天后：忘记 74%
- 1周后：忘记 77%

所以你考前突击记住的东西，考完一周基本忘光了。这不是你记性差，是人类大脑的生理机制。

## 间隔重复的魔法

如果在"快忘记"的时候复习，会发生什么？

```
记忆保留率
100% │●        ●          ●             ●                ●
     │ ╲      /│╲        /│╲           /│╲              /│
 80% │  ╲    / │ ╲      / │ ╲         / │ ╲            / │
     │   ╲  /  │  ╲    /  │  ╲       /  │  ╲          /  │
 60% │    ╲/   │   ╲  /   │   ╲     /   │   ╲        /   │
     │    ●    │    ╲/    │    ╲   /    │    ╲      /    │
 40% │  复习1  │    ●     │     ╲ /     │     ╲    /     │
     │         │  复习2   │      ●      │      ╲  /      │
 20% │         │          │    复习3    │       ╲/       │
     │         │          │             │       ●        │
  0% └─────────┴──────────┴─────────────┴──────复习4─────▶ 时间
     第1天     第2天       第4天         第7天      第15天
```

**神奇的事情发生了**：
- 每次复习后，遗忘速度变慢
- 复习间隔可以越来越长
- 最终变成"永久记忆"

## 间隔重复的参考时间表

```
┌─────────────────────────────────────────────────────────┐
│                   间隔复习时间表（参考）                    │
└─────────────────────────────────────────────────────────┘

  第1次复习        第2次复习        第3次复习        第4次复习        第5次复习
  ┌──────┐        ┌──────┐        ┌──────┐        ┌──────┐        ┌──────┐
  │ 学完 │        │  1天 │        │  3天 │        │  7天 │        │ 15天 │
  │ 当天 │  ───▶  │  后  │  ───▶  │  后  │  ───▶  │  后  │  ───▶  │  后  │
  └──────┘        └──────┘        └──────┘        └──────┘        └──────┘
                                                                      │
                                                                      ▼
                                                               进入长期记忆
```

**简化版口诀**：1-3-7-15（天）

> **重要说明**：上述时间表是一个通用的参考起点，实际最优的复习间隔**因人而异、因材料而异**。有些人可能需要更短的间隔，有些人可以间隔更长。建议：
> 1. 从这个时间表开始尝试
> 2. 根据自己的遗忘情况调整：如果复习时完全忘了，说明间隔太长；如果复习时轻松记得，说明可以延长间隔
> 3. 使用 Anki 等工具可以自动根据你的表现个性化调整间隔（软件内置智能算法，会根据你每次的回答自动计算下次复习时间）

## 为什么间隔重复有效？

### 1. 提取练习效应

每次"想起来"，大脑都在加强这条记忆路径。就像森林里的小路：

- 只走一次 → 很快被杂草覆盖
- 隔段时间走一次 → 路越来越清晰

### 2. 睡眠巩固

学习后睡一觉，大脑会在睡眠中"整理"今天学的东西。第二天复习，正好巩固这个整理结果。

**睡眠对记忆的重要性**：
- 睡眠期间大脑会重播白天学习的内容，强化神经连接
- 深度睡眠阶段对陈述性记忆（如知识、事实）特别重要
- 建议：学习新内容后保证充足睡眠，不要熬夜突击

### 3. 难度适中

太早复习 → 太容易，没挑战
太晚复习 → 忘光了，等于重学

在"快忘记"时复习 → **刚好有点挑战，记得最牢**

## 实战工具推荐

### 1. Anki（推荐，免费神器）

Anki 会自动计算你每张卡片的复习时间：

```
┌─────────────────────────────────┐
│  Q: 费曼学习法的核心是什么？      │
│                                 │
│         [显示答案]               │
└─────────────────────────────────┘
                │
                ▼
┌─────────────────────────────────┐
│  A: 用简单的话教给别人           │
│                                 │
│  [忘了]  [模糊]  [记得]  [太简单] │
│  1分钟   10分钟   1天     4天    │
└─────────────────────────────────┘
```

- 点"忘了"→ 一会儿再问你
- 点"记得"→ 明天再问你
- 点"太简单"→ 过几天再问你

**Anki 快速入门**：

1. **下载安装**：
   - 访问 [官网 apps.ankiweb.net](https://apps.ankiweb.net)
   - 桌面版（Windows/Mac/Linux）：免费
   - iOS 版：付费（约 ¥163）
   - Android 版（AnkiDroid）：免费

2. **创建你的第一个卡组**：
   - 打开 Anki，主界面点击底部的 **"创建牌组"**
   - 输入名称，如"英语单词"或"编程概念"
   - 点击确定

3. **添加第一张卡片**：
   - 点击主界面顶部的 **"添加"** 按钮
   - **正面（Front）**：输入问题，如"什么是费曼学习法？"
   - **背面（Back）**：输入答案，如"用简单的话教给别人，检验是否真正理解"
   - 点击 **"添加"** 保存，可以继续添加更多卡片

4. **开始每日复习**：
   - 回到主界面，点击你创建的卡组
   - 软件会显示今天需要复习的卡片数量
   - 点击 **"现在学习"** 开始
   - 看到问题后先尝试回忆，再点击 **"显示答案"**
   - 根据记忆情况选择：重来/困难/良好/简单

5. **坚持的关键**：每天固定时间打开 Anki 完成当天的复习任务（通常只需5-15分钟）

**进阶技巧**：可以导入他人制作的共享牌组，搜索 "Anki shared decks" 找到大量资源

## 常见误区

### 误区1：复习=重新看一遍

❌ 打开书重新看
✅ 合上书，尝试回忆，想不起来再看

（这就是"主动回忆"，下一节会详细讲）

### 误区2：记住了就不用复习了

❌ "我都记住了，不用复习了"
✅ 记住了更要复习，否则很快就会忘

### 误区3：一次背到滚瓜烂熟

❌ 一个单词重复背100遍
✅ 背10遍，明天再背10遍，效果更好

## 实战练习

### 练习：制定一个间隔复习计划

假设今天是周一，你学了5个新概念：

1. 写下这5个概念

|-----|---------|------|------|------|------|
| ___ | 周一 | 周二 | 周四 | 下周一 | 两周后 |

## 常见问题

### Q1：要复习的东西太多了怎么办？

**答**：这正是 Anki 这类工具的价值——它会自动分配每天的复习量，确保不会堆积。建议每天新学内容控制在可管理范围内。

### Q2：忘记复习了怎么办？

**答**：没关系，补上就行。但尽量不要连续漏掉，否则之前的努力可能白费。

### Q3：什么内容适合用间隔重复？

**答**：需要"记住"的内容：
- 单词、公式、定义
- 历史事件、人名地名
- 编程语法、API
- 任何需要"背"的东西

### Q4：理解性的内容也要用吗？

**答**：理解性内容用费曼学习法，记忆性内容用间隔重复。两者结合最佳。

### Q5：间隔时间必须严格遵守吗？

**答**：1-3-7-15是参考值，±1天问题不大。关键是"间隔"和"重复"这两个原则。实际上，现代间隔重复软件会根据你的表现动态调整，比固定时间表更有效。
