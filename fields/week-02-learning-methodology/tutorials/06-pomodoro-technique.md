# 番茄工作法 (Pomodoro Technique)

## 一句话定义

**25分钟专注 + 5分钟休息，用时间盒子管理注意力。**

## 生活类比

跑马拉松：

- ❌ **错误方式**：一口气冲刺，跑5公里就累趴下了
- ✅ **正确方式**：匀速跑，每5公里补个水，最终跑完全程

学习也是马拉松，番茄工作法就是你的"补给站"。

### 问题2：帕金森定律

> "工作会膨胀以填满可用的时间。"

- 给你一整天写报告 → 拖到最后一刻
- 给你25分钟写初稿 → 高效完成

## 番茄工作法的标准流程

```
┌─────────────────────────────────────────────────────────┐
│                    一个番茄周期                          │
└─────────────────────────────────────────────────────────┘

  ┌──────────────────────────────┐    ┌─────────────────┐
  │         25分钟专注            │───▶│    5分钟休息    │
  │                              │    │                 │
  │  • 只做一件事                 │    │  • 离开座位     │
  │  • 不看手机                  │    │  • 喝水/伸展    │
  │  • 不回消息                  │    │  • 看看远处     │
  │  • 不切换任务                │    │  • 不看手机!    │
  │                              │    │                 │
  └──────────────────────────────┘    └─────────────────┘
                │                            │
                └──────── 重复 4 次 ──────────┘
                             │
                             ▼
                    ┌─────────────────┐
                    │   15-30分钟      │
                    │   长休息         │
                    │                 │
                    │  • 散步         │
                    │  • 吃点东西     │
                    │  • 完全放松     │
                    └─────────────────┘
```

### 具体步骤

1. **选择任务**：决定接下来25分钟要做什么
2. **设置计时器**：25分钟
3. **专注工作**：直到计时器响
4. **短休息**：5分钟
5. **重复**：完成4个番茄后，长休息15-30分钟

## 番茄钟期间的规则

### 规则1：一个番茄不可分割

```
❌ 错误：做了15分钟，去回个消息，然后继续做10分钟
✅ 正确：25分钟内不管发生什么都不中断
```

如果被打断了，这个番茄作废，重新开始一个。

### 规则2：一个番茄只做一件事

```
❌ 错误：写5分钟代码，看10分钟文档，再写10分钟代码
✅ 正确：这25分钟只写代码，下一个番茄再看文档
```

### 规则3：休息时真的休息

```
❌ 错误的休息：刷微博、看视频、回消息
✅ 正确的休息：喝水、伸展、上厕所、看窗外
```

关键是让大脑"换换脑子"，而不是用另一种方式继续消耗注意力。

## 处理打断的方法

### 内部打断（自己想到的事）

突然想起"要回个邮件"或"等会查个东西"：

```
┌─────────────────────────────────────────┐
│  处理内部打断                            │
├─────────────────────────────────────────┤
│  1. 在纸上记下来（3秒）                  │
│  2. 继续当前任务                         │
│  3. 番茄结束后再处理                      │
└─────────────────────────────────────────┘
```

### 外部打断（别人找你）

同事过来问问题、手机来电：

```
┌─────────────────────────────────────────┐
│  处理外部打断                            │
├─────────────────────────────────────────┤
│  方法1：延迟                             │
│  "我25分钟后找你，可以吗？"              │
│                                         │
│  方法2：快速处理（<1分钟）               │
│  如果真的很紧急且能快速解决              │
│                                         │
│  方法3：作废这个番茄                      │
│  如果必须现在处理                        │
└─────────────────────────────────────────┘
```

## 进阶技巧

### 技巧1：估算番茄数

开始一天前，估算每个任务需要几个番茄：

| 任务 | 预估番茄数 | 实际番茄数 |
|-----|----------|----------|
| 写周报 | 2 | |
| 学习新API | 4 | |
| 修复Bug | 3 | |

这样可以更好地规划时间，也能追踪自己的效率。

### 技巧2：调整时间长度

标准是25+5，但可以根据个人情况调整：

```
初学者：      15分钟专注 + 3分钟休息
标准：        25分钟专注 + 5分钟休息
心流模式：    50分钟专注 + 10分钟休息
```

关键是找到适合自己的节奏。

### 技巧3：番茄日志

记录每天完成的番茄数和对应任务：

```
日期：2024-01-15
────────────────────────────────────
🍅🍅🍅🍅 - 写文档
🍅🍅 - 代码审查  
🍅🍅🍅 - 学习新框架
🍅 - 回复邮件
────────────────────────────────────
总计：10个番茄
```

## 常见问题

### Q1：25分钟太短了，刚进入状态就要休息？

**答**：如果已经进入心流状态，可以：
- 继续做完，不强制休息
- 或者延长到50分钟一个番茄

番茄钟是帮助你开始专注的工具，不是限制你的牢笼。

### Q2：任务没做完，25分钟就到了怎么办？

**答**：记下进度，休息后从断点继续。很多任务需要多个番茄才能完成，这很正常。

### Q3：有些任务需要持续思考，能不能不休息？

**答**：可以，但要注意：
- 大脑需要休息才能保持效率
- "继续思考"和"大脑疲劳"有时很难区分
- 至少每2小时强制休息一次

### Q4：感觉番茄工作法让我变机械了？

**答**：番茄钟是工具，不是目的。
- 不需要每分每秒都用番茄钟
- 创意工作可以更灵活
- 适合需要"强制专注"的任务

### Q5：休息时总忍不住想看手机？

**答**：
- 把手机放到另一个房间
- 休息时做需要移动的事（喝水、上厕所）
- 设置手机的专注模式
